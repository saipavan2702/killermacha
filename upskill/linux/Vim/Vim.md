## Help and Manual
```vim
K                " Open help for word under cursor
:help <cmd>      " Help for command
:help user-manual
```

---
## Spell Check Shortcuts
```vim
]s / [s         " Next / previous misspelled word
z=              " Show suggestions
zg              " Add word to dictionary
zw              " Mark word as incorrect
```
